Instructions for the Game!

  1. Press an empty Square and enter a number.
  2. The number will be entered as a PROBABLE.
  3. To confirm the number and add it permanently select it and PRESS ENTER.
  4. If the number was at a valid position, the number will become part of the board.
  5. If the number was invalid, the number will be erased.
  6. Only three negative entries are allowed. The game will be over after that.
